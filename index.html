SIC3D: Style Image Conditioned Text-to-3D Gaussian Splatting Generation User Study

Please evaluate 3D generation models by comparing renderings

Task Overview

The task is to compare two 3D models, both generated from similar input information. The input contains two kinds of information: text prompt, style image. Objects are generated using these two pieces of information with different models. You must decide which one is better according to the provided criteria.

Image Layout

I will provide you with a multiple images from different views divided into two rows and four columns, each of which contains an image. The four images on the left are four renderings of Method A's result, and the four images on the right are four renderings of Method B's result. Examine the multi-part image carefully. Attempt to mentally construct a 3D model of object 1 using the 4-view rendering in the left part. Attempt to mentally construct a 3D model of object 2 using the 4-view rendering in the right part.

Evaluation Criteria

For each sample you will answer two questions.

1. Overall Object Quality

Decide which of the two 3D objects is better overall. Do not rely on a single property — form your judgement by considering all of the following aspects together:

  • Text–Asset Alignment — how well the object matches the text prompt and captures the attributes it describes.
  • 3D Plausibility — natural, solid and plausible, with accurate proportions and no abnormalities (duplicated parts, noisy or meaningless structures).
  • Geometry–Texture Alignment — the texture adheres to the geometry and the two align with each other locally.
  • Low-Level Texture Details — fine texture details that are sharp rather than blurry.
  • Low-Level Geometry Details — an accurate, well-organised shape with intricate detail and no meaningless noise.

Weigh these aspects together and choose the object that is better overall.

2. Style Alignment

Both objects are generated from the same base text description and influenced by the style feature of the reference image. Decide which object has a style more similar to the reference image — an ideal rendering should have strong commonality with the reference, especially in the colour palette and continuity of tones. For example, if the reference contains fire with intense orange-red tones, the desired object should share a similar colour and tone, as if it were also under fire. Subtle differences in shape and detailed textures are acceptable as long as the overall visual effect is consistent.

Make a Decision

For each of the criteria, you need to make a decision using these three options:

  • Method A is better - The result for Method A is better
  • Method B is better - The result for Method B is better
  • Cannot decide - Both are equal

IMPORTANT: PLEASE USE "Cannot decide" SPARSELY.
