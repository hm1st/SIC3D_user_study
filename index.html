User Study: Stylized Text-to-3D Generation Comparison

Please evaluate 3D generation models by comparing renderings

Task Overview

The task is to compare two 3D models, both generated from similar input information. The input contains two kinds of information separated by a dash: content information - style attribute. The object is generated using these two pieces of information. You must decide which one is better according to the provided criteria.

Image Layout

I will provide you with a multi-part image divided into two rows and four columns, each of which contains an image. The four images on the left are four renderings of object 1 from different viewpoints, and the four images on the right are four renderings of object 2 from different viewpoints. Examine the multi-part image carefully. Attempt to mentally construct a 3D model of object 1 using the 4-view rendering in the left part. Attempt to mentally construct a 3D model of object 2 using the 4-view rendering in the right part.

Evaluation Criteria

1. Text prompt and Model Alignment

Focus on how well the two 3D models correspond to the given text input. An ideal model should accurately reflect what is mentioned in the text prompt, capturing the corresponding attributes as described. Please first describe each of the two models, and then evaluate how well it covers all the attributes in the original text prompt.

2. 3D Plausibility

Determine which of the two 3D models appears more natural, solid, and plausible. Pay attention to any irregularities, such as abnormal body proportions, duplicated parts, or the presence of noisy or meaningless 3D structures. An ideal model should possess accurate proportions, shapes, and structures that closely resemble the real-world object.

3. Geometry-Texture Alignment

This examines how well the texture adheres to the geometry. The texture and shape should align with each other locally.

4. Low-Level Texture Details

Focus on local parts of the renderings. RGB images. Assess which 3D model effectively captures fine details without appearing blurry and which one aligns with the desired aesthetic of the 3D model.

5. Low-Level Geometry Details

The geometry of the 3D models should accurately represent the intended shape. Note that meaningless noise is not considered as high-frequency details. Determine which one has a more well-organized and efficient structure, which one exhibits intricate details, and which one is more visually pleasing and smooth.

6. Overall Quality

Considering all the degrees above, which one is better overall?

7. Style Alignment (for stylized generation)

Our task is to compare two 3D models to see if they match the style of a reference image. Both the 3D models are generated from the same base text description and influenced by the style feature of the reference image.

For each 3D model, determine which one has a similar style to the reference image. An ideal model's rendering should have strong commonality with reference image, especially in the colour palette and continuity of tones. For example, the reference image contains fire with intense orange-red tones. Desired object's rendering should also have a similar color and tone as if it's also under fire. While there are subtle differences in flame shape and detailed textures, the overall visual effect is very consistent.

Output Format

For each of the criteria, you need to make a decision using these three options:

  • Left is better - Left (object 1) is better
  • Right is better - Right (object 2) is better
  • Cannot decide - Both are equal

IMPORTANT: PLEASE USE "Cannot decide" SPARSELY.
